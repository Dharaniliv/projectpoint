# ProjectPoint ‚Äì Real Estate Discovery MVP

ProjectPoint is a simple MVP website built to validate a real estate discovery platform where users can explore real estate projects by city and state, view project details, and connect with builders. Builders can submit their projects, and admins can review and approve listings before they go live.
